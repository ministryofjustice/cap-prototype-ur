# E2E Tests

> These tests were inherited from the original CAP repo and target the Express server in `server/`, not the static GitHub Pages prototype in `docs/`.
>
> They are not actively maintained for UR prototype work.

## Running (against the Express server)

```bash
# Start the server
npm run start:dev

# Run all e2e tests
npm run e2e

# Run with UI
npm run e2e:ui
```
